HTML

  • The head element contains information (metadata) about the webpage
  • The body element represent the visible content shown to the user
  • Visisble HTML elements are contained in between an opening and closing tag
  • A sementic element describes the content and role it has on the webpage (eg:form, article)
  • The 'p' elements reppresents a block of text
  • An image element can contain various attributes (eg:location, description)
  • The anchor('a')element creates links to webpages

CSS

  • A margin property indicates how much space we want around the outside of an element
  • Padding property adds space around the inside of the element
  • To apply a rule to all elements visible on the page use *
  • You can refer to colours using hexadecimal code (hex #)
  • Any HTML element assigned to the class will gain these property assignments from the CSS

Git

  • git status: checks what branch wer are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git pull down origin main: to check that local branch is in synch with thr base branch in GitHub
  • git add -A: adds all modifications in the current working branch to the staging area
  • The staging area informs git that changes will be added in the next commit
  • git commit -m "wanted message": commits changes with a messages that describes the changes
